# 《Claude Cowork：31 個超好用範例實作》— 章節互動練習

這個資料夾放**依章節設計的互動練習**，一章一個子資料夾。每章根據書中內容 + 書中範例，
做成「開瀏覽器就能互動、即時看 demo」的自足網頁（單一 `index.html`，不需安裝、不需連網）。

## 使用方式

1. 進到某一章資料夾（例如 `ch01/`）。
2. 用瀏覽器打開該資料夾裡的 `index.html`（直接雙擊，或右鍵 → 用瀏覽器開啟）。
3. 跟著練習互動：讀重點 → 做題目 → 看即時對錯與解釋 → 到書中對應的 Claude Cowork 介面實際動手做一次。

## 設計原則

- **繁體中文**、深色主題，比照本專案其他 viewer 風格。
- **單檔自足**：HTML 內含 CSS/JS，無外部相依，可離線開。
- **題目來自書本**：情境與判準取自各章「適合場景 / 範例」，不是憑空杜撰。
- **練習 → 真實 demo**：每章尾端給一個「到 Claude Cowork 實際做一次」的建議任務，把練習接回真實操作。
